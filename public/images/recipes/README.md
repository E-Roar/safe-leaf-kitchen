
# Recipe Images

This directory contains recipe images for the SafeLeafKitchen app.

## Image Naming Convention

Recipe images should be named exactly as the English recipe title, converted to lowercase with spaces replaced by hyphens and special characters removed.

### Example Conversions:

| Recipe Title (English) | Image Filename |
|------------------------|----------------|
| Stuffed Msemen with Onion Leaves | `stuffed-msemen-with-onion-leaves.png` |
| Barley Flatbread with Onion Leaves | `barley-flatbread-with-onion-leaves.png` |
| Omelette with Onion Leaves | `omelette-with-onion-leaves.png` |
| Powdered Dried Onion Leaves | `powdered-dried-onion-leaves.png` |

## Image Requirements

- **Format**: PNG
- **Aspect Ratio**: 16:9 or 4:3 recommended
- **Resolution**: Minimum 800x600px, recommended 1200x900px
- **File Size**: Keep under 500KB for optimal loading

If an image is not found, the app will display a placeholder with a chef hat icon.
