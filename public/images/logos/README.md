# Logos Directory

This directory contains logos for partners, sponsors, and organizations.

## File Organization Structure

### Partner Logos
- `fao.png` - FAO (Food and Agriculture Organization) logo
- `fao.webp` - FAO logo optimized for web
- `fao.ico` - FAO favicon format

- `brightidea.png` - BrightIdea logo  
- `brightidea.webp` - BrightIdea logo optimized for web
- `brightidea.ico` - BrightIdea favicon format

- `innovation-hub.png` - Innovation Hub logo
- `innovation-hub.webp` - Innovation Hub logo optimized for web  
- `innovation-hub.ico` - Innovation Hub favicon format

- `universites.png` - Universities consortium logo
- `universites.webp` - Universities logo optimized for web
- `universites.ico` - Universities favicon format

## Format Guidelines

### PNG Files
- Use for high-quality logos with transparency
- Recommended size: 200x200px or 300x300px
- Suitable for print and high-resolution displays

### WebP Files  
- Use for optimized web performance
- Smaller file sizes with good quality
- Recommended for landing page display

### ICO Files
- Use for favicons and small icon representations
- Multiple sizes embedded (16x16, 32x32, 48x48)
- Browser compatibility for icon displays

## Usage in Code

Logos are referenced in the acknowledgments section using:
```jsx
<img src="/images/logos/fao.webp" alt="FAO" />
```

## Future Additions

When adding new partner logos:
1. Follow the naming convention: `partner-name.format`
2. Provide all three formats (png, webp, ico)
3. Update the code references accordingly
4. Ensure logos are properly licensed for use
